BLOCCO A — URTO (parola chiave: "urto", "collisione", "proiettile conficcato", "si attaccano", "si scontrano")
⚠ In un urto, la quantità di moto TOTALE si conserva SEMPRE (se non ci sono forze esterne). Chiediti: l'urto è elastico o anelastico?
Step 1 — Tipo di urto?
Anelastico (rimangono uniti / "conficcato" / non specificato)
Elastico (specificato esplicitamente)
2D / obliquo (angoli dati)
URTO ANELASTICO (o completamente anelastico)
Legge: conservazione quantità di moto SOLO
m₁v₁ᵢ + m₂v₂ᵢ = m₁v₁f + m₂v₂f
Se completamente anelastico (rimangono uniti → v_f comune):
m₁v₁ᵢ + m₂v₂ᵢ = (m₁ + m₂)·vf
→ vf = (m₁v₁ᵢ + m₂v₂ᵢ) / (m₁ + m₂)
Se v₂ᵢ = 0 (bersaglio fermo):
vf = m₁·v₁ᵢ / (m₁ + m₂)
Energia persa nell'urto:
ΔKurto = ½(m₁+m₂)vf² - ½m₁v₁ᵢ² - ½m₂v₂ᵢ² (< 0)
Output tipici: vf subito dopo l'urto, ΔK persa
⚠ Se dopo l'urto c'è una molla o un'oscillazione → usa il risultato vf nel BLOCCO ENERGIA
→ Continua nel BLOCCO ENERGIA
↑ chiudi
URTO ELASTICO (1D)
Leggi: conservazione p + conservazione K
m₁v₁ᵢ + m₂v₂ᵢ = m₁v₁f + m₂v₂f
½m₁v₁ᵢ² + ½m₂v₂ᵢ² = ½m₁v₁f² + ½m₂v₂f²
Formule chiuse (combina le due equazioni):
v₁f = [(m₁-m₂)v₁ᵢ + 2m₂v₂ᵢ] / (m₁+m₂)
v₂f = [2m₁v₁ᵢ + (m₂-m₁)v₂ᵢ] / (m₁+m₂)
Caso speciale — bersaglio fermo (v₂ᵢ = 0):
v₁f = (m₁-m₂)/(m₁+m₂) · v₁ᵢ
v₂f = 2m₁/(m₁+m₂) · v₁ᵢ
Se m₁ = m₂: scambio di velocità!
v₁f = 0; v₂f = v₁ᵢ
Se m₂ ≫ m₁ (parete rigida):
v₁f ≈ -v₁ᵢ (rimbalza indietro)
Output tipici: v₁f, v₂f, variazione di K (= 0 se elastico)
→ Verifica con BLOCCO ENERGIA se richiesto
↑ chiudi
URTO 2D (due dimensioni)
Legge: conservazione p in forma vettoriale → scomponi per asse
asse x: m₁v₁ᵢ + m₂v₂ᵢ = m₁v₁f·cosθ₁ + m₂v₂f·cosθ₂
asse y: 0 = -m₁v₁f·sinθ₁ + m₂v₂f·sinθ₂
Se elastico, aggiungi:
½m₁v₁ᵢ² + ½m₂v₂ᵢ² = ½m₁v₁f² + ½m₂v₂f²
⚠ 4 incognite (v₁f, v₂f, θ₁, θ₂) → servono 3 equazioni + 1 dato aggiuntivo (uno degli angoli o una velocità finale)
Output tipici: velocità e angoli di uscita
↑ chiudi
↓
BLOCCO B — ENERGIA (parola chiave: "velocità in un punto", "quota", "molla compressa/estesa", "attrito", "lavoro", "distanza percorsa")
⚠ Prima di scrivere l'equazione, identifica: ci sono forze non conservative (attrito)? Ci sono molle?
Step 1 — Forze dissipative?
No attrito → energia meccanica CONSERVATA
Sì attrito (µ dato, o "su piano ruvido")
C'è una MOLLA (k dato, compressione x)
ENERGIA MECCANICA CONSERVATA (no attrito)
Principio: Eᵢ = Ef
½mv²ᵢ + mghᵢ + ½kx²ᵢ = ½mv²f + mghf + ½kx²f
Versione senza molla:
½mv²ᵢ + mghᵢ = ½mv²f + mghf
Se parte da fermo (vᵢ = 0):
mghᵢ = ½mv²f + mghf → v = √[2g(hᵢ-hf)]
Se hf = 0 (riferimento al suolo):
v = √(2gh)
Con molla, da fermo (Eᵢ = ½kx²):
½kx² = ½mv² → v = x·√(k/m)
Output tipici: velocità in un punto, quota massima, compressione molla
↑ chiudi
ENERGIA CON ATTRITO (forze non conservative)
Principio: ΔEmecc = lavoro forze non conservative
ΔK + ΔU = L_nc
(Kf - Kᵢ) + (Uf - Uᵢ) = -fk·ℓ
Forma esplicita:
½mv²f - ½mv²ᵢ + mg(hf-hᵢ) = -µk·N·ℓ
Su piano orizzontale (h costante):
½mv²f = ½mv²ᵢ - µk·mg·ℓ
Su piano inclinato (angolo θ, N = mg cosθ):
v = √[2gℓ(sinθ - µk·cosθ)] (partendo da fermo)
Distanza di arresto (vf = 0, vᵢ nota):
ℓ = v²ᵢ / (2µk·g) (piano orizzontale)
Forza di attrito dinamico:
fk = µk · N
Forza normale su piano inclinato:
N = mg·cosθ
⚠ L'attrito statico vale: fs ≤ µs·N (non fa lavoro se il corpo non si muove)
Output tipici: velocità finale, distanza percorsa, energia dissipata = µk·N·ℓ
↑ chiudi
MOLLA (legge di Hooke + energia elastica)
Forza molla:
F = -k·x (k = costante elastica [N/m], x = spostamento da riposo)
Energia potenziale elastica:
U_el = ½·k·x²
Energia totale con molla (no attrito):
½mv² + mgh + ½kx² = costante
Velocità massima (in x=0, punto di equilibrio):
½mv²max = ½kA² → vmax = A·√(k/m)
Ampiezza di oscillazione:
A = xmax (compressione/allungamento massimo)
Frequenza propria:
ω = √(k/m); T = 2π/ω = 2π·√(m/k)
⚠ Se prima c'è un urto: usa vf dell'urto come vᵢ per la molla → poi bilancio energetico
Output tipici: ampiezza oscillazione, vmax, compressione x
← Combina con BLOCCO URTO (pendolo balistico)
↑ chiudi
↓
BLOCCO C — CINEMATICA (parola chiave: "proiettile", "lanciato", "traiettoria", "gittata", "tempo di volo", "moto rettilineo uniforme/accelerato")
Moto parabolico (v₀ e angolo α dati)
Moto rettilineo (a = cost, o MRU)
Moto circolare
MOTO PARABOLICO
Scomponi SEMPRE il moto in x (costante) e y (accelerato):
vx = v₀·cosα (costante)
vy(t) = v₀·sinα - g·t
x(t) = v₀·cosα · t
y(t) = y₀ + v₀·sinα · t - ½g·t²
Tempo al massimo (vy = 0):
t_max = v₀·sinα / g
Quota massima:
H = y₀ + (v₀·sinα)² / (2g)
Gittata (da y=y₀, stesso piano):
R = v₀²·sin(2α) / g
Se lanciato orizzontalmente (α = 0):
x(t) = v₀·t; y(t) = h - ½g·t²
t_caduta = √(2h/g); x_caduta = v₀·√(2h/g)
Velocità in un punto generico:
|v| = √(vx² + vy²); θ = arctan(vy/vx)
Output tipici: gittata R, quota max H, tempo di volo, posizione in t, velocità finale
↑ chiudi
MOTO RETTILINEO UNIFORMEMENTE ACCELERATO (MRUA)
v(t) = v₀ + a·t
x(t) = x₀ + v₀·t + ½a·t²
v² = v₀² + 2a·(x - x₀)
Caduta libera (a = g = 9.81 m/s²):
v = g·t; h = ½g·t²; v = √(2gh)
MRU (a = 0):
x(t) = x₀ + v·t
Output tipici: posizione, velocità, tempo, accelerazione
↑ chiudi
MOTO CIRCOLARE UNIFORME
v = ω·r; ω = 2π/T = 2π·f
a_c = v²/r = ω²·r (centripeta, diretta verso il centro)
Forza centripeta necessaria:
F_c = m·v²/r = m·ω²·r
Su curva stradale (con attrito):
µs·mg ≥ m·v²/r → v_max = √(µs·g·r)
Output tipici: v, ω, T, f, raggio min, forza centripeta
↑ chiudi
↓
BLOCCO D — DINAMICA (parola chiave: "forza", "tensione", "fune", "piano inclinato", "accelerazione", "equilibrio")
⚠ PROCEDURA: (1) Disegna il diagramma delle forze. (2) Scegli assi comodi. (3) Scrivi ΣF = ma per ogni asse. (4) Risolvi il sistema.
Equilibrio (a = 0)
Accelerazione (a ≠ 0)
Piano inclinato
Fune / Carrucola
EQUILIBRIO STATICO
ΣF = 0 → Fx = 0 e Fy = 0
Corpo appeso a funi con angoli θ₁, θ₂:
T₁·cosθ₁ = T₂·cosθ₂ (asse x)
T₁·sinθ₁ + T₂·sinθ₂ = Mg (asse y)
Piano inclinato (no attrito, fune parallela al piano):
T = mg·sinθ; N = mg·cosθ
Output tipici: tensioni, forza normale, angoli
↑ chiudi
SECONDA LEGGE DI NEWTON
ΣF = m·a (forma vettoriale)
ΣFx = m·ax; ΣFy = m·ay
Sistema con due masse collegate (fune inestensibile):
a = (m₂·g) / (m₁ + m₂) (m₂ appeso, m₁ su piano senza attrito)
T = m₁·m₂·g / (m₁ + m₂)
Ascensore (N = reazione bilancia):
N - mg = ma → N = m(g + a) [accelera in su]
N = m(g - a) [accelera in giù]
Output tipici: accelerazione, tensione, forza normale
↑ chiudi
PIANO INCLINATO (angolo θ)
Asse x ∥ piano (positivo verso il basso), asse y ⊥ piano:
N = mg·cosθ
a = g·sinθ (senza attrito)
a = g·(sinθ - µk·cosθ) (con attrito)
Condizione di scivolamento (attrito statico):
tanθ > µs → il corpo scivola
Output tipici: N, accelerazione, µ_min, tensione fune
↑ chiudi
FUNE / CARRUCOLA (fune di massa trascurabile)
Regola: la tensione è uguale ovunque nella fune (se fune ideale).
T = costante lungo la fune
Sistema Atwood (due masse appese):
a = (m₁ - m₂)·g / (m₁ + m₂)
T = 2·m₁·m₂·g / (m₁ + m₂)
Output tipici: tensione T, accelerazione a
↑ chiudi
↓
BLOCCO E — GRAVITAZIONE (parola chiave: "orbita", "satellite", "pianeta", "velocità di fuga", "periodo orbitale", G dato)
G = 6.674 × 10⁻¹¹ N·m²/kg²
Forza gravitazionale:
F = G·M·m / r²
Energia potenziale gravitazionale:
U = -G·M·m / r
Orbita circolare (F_g = F_c):
G·M·m/r² = m·v²/r → v = √(G·M/r)
Periodo orbitale (v = 2πr/T):
T = 2π·r / v = 2π·√(r³ / G·M)
Terza legge di Keplero:
T² = (4π²/G·M) · r³ → T² ∝ r³
Velocità di fuga (U + K = 0 a r → ∞):
½mv²_fuga = G·M·m/r → v_fuga = √(2G·M/r)
Energia totale orbita circolare:
E_tot = K + U = ½mv² - G·M·m/r = -G·M·m/(2r)
Per confrontare due orbite (conservazione energia):
-G·M·m/(2r₁) = -G·M·m/(2r₂) + ΔE
Output tipici: v orbitale, T, r, v_fuga, energia orbita
↓
BLOCCO F — ROTAZIONE / CORPO RIGIDO (parola chiave: "ruota", "cilindro", "momento d'inerzia", "rotola", "asta", "momento angolare")
Energia cinetica di rotazione:
K_rot = ½·I·ω²
Energia totale (traslazione + rotazione):
K_tot = ½mv² + ½Iω²
Puro rotolamento (v = ω·R):
K_tot = ½mv² + ½I(v/R)² = ½v²(m + I/R²)
Momenti d'inerzia principali:
Cilindro solido (asse): I = ½mR²
Sfera solida: I = ⅖mR²
Asta (asse al centro): I = 1/12·mL²
Asta (asse a un'estrem): I = ⅓mL²
Anello / guscio cil.: I = mR²
Teorema degli assi paralleli:
I = I_cm + m·d² (d = distanza dal cm)
Seconda legge angolare:
Στ = I·α (τ = r × F = r·F·sinφ)
Conservazione momento angolare (Στ_ext = 0):
L = I·ω = costante
I₁·ω₁ = I₂·ω₂
Output tipici: ω, α, v_cm, velocità al fondo di un piano inclinato con rotolamento
⚠ Per corpo che rotola su piano inclinato: usa conservazione energia con K_tot (includi K_rot)
LEGENDA: ⚡ = nodo decisionale | → = collegamento tra blocchi | ΔK = variazione energia cinetica | ΔU = variazione energia potenziale | vᵢ = velocità iniziale | vf = velocità finale | ℓ = distanza | g = 9.81 m/s²
